# Build Resources

This directory holds assets used by `electron-builder` during packaging.

`icon.png` — 1024×1024 master icon. electron-builder auto-derives .icns
(macOS) and .ico (Windows) from this file at build time. The current
`icon.png` is a placeholder "L" generated by
`generate-placeholder-icon.mjs` and should be replaced with branded
artwork before public release.

To regenerate the placeholder:

    node build/generate-placeholder-icon.mjs

Optional files (not yet present):

- `background.png` — macOS DMG background
- `entitlements.mac.plist` — macOS hardened-runtime entitlements (only
  needed when enabling notarization; set `hardenedRuntime: true` in
  electron-builder.yml first)

See [electron-builder docs: Icons](https://www.electron.build/icons).
